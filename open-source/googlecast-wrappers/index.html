Google Cast Wrappers

View this project on GitHub

Overview

Simple wrappers around the core Google Cast Sender and Receiver SDKs that allow for quick integration into new, or existing, projects.

Goals

  • Gain a deeper understanding of the Google Cast SDK and how best to take advantage for it for current and future projects.
  • Simplify the setup for Google Cast Chrome Sender and Receiver apps.
  • Make both the Sender and Receiver setup abstract enough to be easily integrated with existing projects/components.
  • Provide a simpler interface for communicating between Sender -> Receiver and Receiver -> Sender.

Examples

  • Sender App - Custom Player Interface (Vimeo)

    A simple interface that allows people to paste in Vimeo video uri into an input field, which plays the video in a custom receiver app, and supplies the sender with status updates.
